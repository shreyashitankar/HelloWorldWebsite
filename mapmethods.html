Map Object Methods

Clear()

Deletes All the elements From the map Object

Syntax

let john = {name: 'John Doe'},
lily = {name: 'Lily Bush'},
peter = {name: 'Peter Drucker'};
let userRoles = new Map();
userRoles.set(john, 'admin')
         .set(lily, 'editor')
         .set(peter, 'subscriber');
userRoles.get(john);
userRoles.clear();
console.log(userRoles.size)

Output

0

entries()

returns a new Iterator object that contains an array of [key, value] for each element in the map object. The order of objects in the map is the same as the insertion order.

let john = {name: 'John Doe'},
lily = {name: 'Lily Bush'},
peter = {name: 'Peter Drucker'};
let userRoles = new Map();
userRoles.set(john, 'admin')
            .set(lily, 'editor')
            .set(peter, 'subscriber');
userRoles.entries();

forEach()

forEach() method calls a function for each element in the array.

let john = {name: 'John Doe'},
            lily = {name: 'Lily Bush'},
            peter = {name: 'Peter Drucker'};
            let userRoles = new Map();
            userRoles.set(john, 'admin')
                        .set(lily, 'editor')
                        .set(peter, 'subscriber');
            userRoles.forEach(function(val,index){
                console.log(val)
            })

Output

VM504:1 admin
VM504:1 editor
VM504:1 subscriber
