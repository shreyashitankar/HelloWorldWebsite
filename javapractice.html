Const Keyword

Constants are block-scoped, much like variables defined using the let keyword. The value of a constant can't be changed through reassignment, and it can't be redeclared.

const number = 42;
          try {
                const number = 99;
              } catch (err) {
                              console.log(err);
                              // expected output: TypeError: invalid assignment to const `number'
                              // Note - error messages will vary depending on browser
              }
            console.log(number);

// expected output: 42

Rest Parameters

ES6 provides a new kind of parameter so-called rest parameter that has a prefix of three dots (...). The rest parameter allows you to represent an indefinite number of arguments as an array. See the following

code:

function sum(...args) {
            let total = 0;
            for (const a of args) {
                total += a;
            }
            return total;
        }
        
        sum(1, 2, 3);

Output Will look like this

//output = 6

The rest parameter should be in last

For __ of __ Loop

ES6 introduced a new construct for ... of that creates a loop iterating over an iterable object such as an array , a map , a set and etc.

syntax 

for (variable of iterable) {
    // statements 
}

Example 

let scores = [80, 90, 70];
for (let score of scores) {
                            score = score + 5;
                            console.log(score);
                          }

Output Will Look Like This

//85
//95

ES6 map Object

A map objects holds key-value paris where values of any type can be used as either keys or values.

A map objects remembers the original insertion order of the keys 

let map = new Map([iterable]);

Example Code

let john = {name: 'John Doe'},
lily = {name: 'Lily Bush'},
peter = {name: 'Peter Drucker'};
let userRoles = new Map();
userRoles.set(jhon, 'admin')
         .set(lily, 'editor')
         .set(peter, 'subscriber');
userRoles.get(john);
