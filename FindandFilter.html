Find And Filter Methods

Find Method

Find Method return the first value which is required

const array1 =  [5, 12, 8, 130, 44];
                const found = array1.find(element => element > 7);
                console.log(found);

Output

12

Here output is 12 right but it should return 8 or 130 or 44 but its only 12 because 12 has 1st occurence

FindIndex Method

const array1 =  [5, 12, 8, 130, 44];
                const found = array1.findIndex(element => element==12);
                console.log(found);
            
1

This will return index of 1st occurence of the number.

Filter() Method

The filter method creates a new array according to the condition or fucntion provided by the programmer.

const words = ['spray','limit', 'elite', 'exuberant', 'destruction', 'present'];
                let result = words.filter(word => word.length > 6);
                console.log(result);

Output

["exuberant", "destruction", "present"]
