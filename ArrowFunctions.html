Arrow functions

What are Arrow Functions

Arrow functions – also called “fat arrow” functions, from CoffeeScript (a transcompiled language) — are a more concise syntax for writing function expressions. They utilize a new token, =>, that looks like a fat arrow. Arrow functions are anonymous and change the way this binds in functions. Arrow functions make our code more concise, and simplify function scoping and the this keyword. They are one-line mini functions which work much like Lambdas in other languages like C# or Python. (See also lambdas in JavaScript). By using arrow functions, we avoid having to type the function keyword, return keyword (it’s implicit in arrow functions), and curly brackets.

Example

const multiplyES6 = (x, y) => { return x * y };
            console.log(multiplyES6(7, 7));

OutPut

49

const phraseSplitterEs6 = phrase => phrase.split(" ");

Example 3 No Parameter

const docLogEs6 = () => { console.log("hello"); };
